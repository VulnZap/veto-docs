Veto works locally out of the box. This tutorial gives you a fully runnable example — copy-paste each step and see Veto allow one tool call and block another.

## 1. Install

<Tabs items={["TypeScript", "Python"]}>
<Tab value="TypeScript">

```bash
npm install veto-sdk
npx veto init
```

</Tab>
<Tab value="Python">

```bash
pip install veto
veto init
```

</Tab>
</Tabs>

This creates a `veto/` directory with your config:

```
veto/
├── veto.config.yaml
└── rules/
    └── default.yaml
```

## 2. Write a rule

Replace the contents of `veto/rules/default.yaml` with:

```yaml
rules:
  - id: limit-transfers
    name: Block large transfers
    action: block
    tools:
      - transfer_funds
    conditions:
      - field: arguments.amount
        operator: greater_than
        value: 1000
```

Any call to `transfer_funds` with `amount > 1000` will be blocked.

## 3. Run it

<Tabs items={["TypeScript", "Python"]}>
<Tab value="TypeScript">

Create `demo.ts` and paste:

```typescript
import { Veto, ToolCallDeniedError } from 'veto-sdk';

const veto = await Veto.init();

const tools = veto.wrap([{
  name: 'transfer_funds',
  handler: async (args: any) => `Transferred $${args.amount} to ${args.to}`,
}]);

// This one passes ($500 is under the $1000 limit)
const result = await tools[0].handler({ amount: 500, to: 'Alice' });
console.log('Allowed:', result);

// This one gets blocked ($5000 exceeds the $1000 limit)
try {
  await tools[0].handler({ amount: 5000, to: 'Bob' });
} catch (error) {
  if (error instanceof ToolCallDeniedError) {
    console.log('Denied:', error.reason);
  }
}

console.log('Stats:', veto.getHistoryStats());
```

Run it:

```bash
npx tsx demo.ts
```

</Tab>
<Tab value="Python">

Create `demo.py` and paste:

```python
import asyncio
from veto import Veto
from veto.core.interceptor import ToolCallDeniedError

async def main():
    veto = await Veto.init()

    tools = veto.wrap([{
        "name": "transfer_funds",
        "handler": lambda args: f"Transferred ${args['amount']} to {args['to']}",
    }])

    # This one passes ($500 is under the $1000 limit)
    result = await tools[0].handler({"amount": 500, "to": "Alice"})
    print("Allowed:", result)

    # This one gets blocked ($5000 exceeds the $1000 limit)
    try:
        await tools[0].handler({"amount": 5000, "to": "Bob"})
    except ToolCallDeniedError as error:
        print("Denied:", error.reason)

    print("Stats:", veto.get_history_stats())

asyncio.run(main())
```

Run it:

```bash
python demo.py
```

</Tab>
</Tabs>

Expected output:

```
Allowed: Transferred $500 to Alice
Denied: Matched rule: Block large transfers
Stats: { totalCalls: 2, allowedCalls: 1, deniedCalls: 1 }
```

That's it. Veto intercepted the tool call, evaluated the rule locally in ~1-5ms, and blocked it. No network calls, no API key.
